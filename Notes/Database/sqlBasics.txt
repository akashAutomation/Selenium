ðŸ‘‰ SQL (Structured Query Language) is not a single language but a family of standardized commands and dialects used
   to interact with relational databases.

1. DDL (Data Definition Language)
   - Defines and manages database structures.
   - Commands: CREATE, ALTER, DROP, TRUNCATE
2. DML (Data Manipulation Language)
   - Handles data inside tables.
   - Commands: INSERT, UPDATE, DELETE, MERGE
3. DQL (Data Query Language)
   - Focuses on retrieving data.
   - Command: SELECT
4. DCL (Data Control Language)
   - Manages permissions and access.
   - Commands: GRANT, REVOKE
5. TCL (Transaction Control Language)
   - Controls transactions and ensures data integrity.
   - Commands: COMMIT, ROLLBACK, SAVEPOINT
