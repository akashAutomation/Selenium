ğŸ‘‰ What Is a Frame?
- A frame is an HTML element (<iframe> or <frame>) that embeds another HTML document within the current page.
- Commonly used for ads, embedded widgets, or third-party content.

ğŸ‘‰ Switching Between Frames
Selenium provides multiple ways to switch into a frame:
1. By Index:
        driver.switchTo().frame(0); // Switch to the first frame on the page

2. By Name or ID:
        driver.switchTo().frame("frameName"); // Use the name or ID attribute

3. By WebElement:
        WebElement frame = driver.findElement(By.tagName("iframe"));
        driver.switchTo().frame(frame); // Switch using a located frame element

ğŸ‘‰ Switching Back to Main Content:
- After interacting with the frame, switch back to the main page:
    driver.switchTo().defaultContent(); // Goes back to the top-level page

- Or to the parent frame (if nested):
    driver.switchTo().parentFrame(); // Goes one level up
